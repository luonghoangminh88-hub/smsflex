# VietQR Bank Codes - Danh sách mã ngân hàng chuẩn

Khi cấu hình payment methods trong admin, sử dụng các bank_code sau:

| Ngân hàng | Bank Code | Tên đầy đủ |
|-----------|-----------|------------|
| Vietcombank | VCB | Ngân hàng TMCP Ngoại Thương Việt Nam |
| MB Bank | MB | Ngân hàng TMCP Quân Đội |
| Techcombank | TCB | Ngân hàng TMCP Kỹ Thương Việt Nam |
| ACB | ACB | Ngân hàng TMCP Á Châu |
| Timo | VCCB | Ngân hàng TMCP Bản Việt (Viet Capital Bank) |
| VPBank | VPB | Ngân hàng TMCP Việt Nam Thịnh Vượng |
| TPBank | TPB | Ngân hàng TMCP Tiên Phong |
| Sacombank | STB | Ngân hàng TMCP Sài Gòn Thương Tín |
| VietinBank | CTG | Ngân hàng TMCP Công Thương Việt Nam |
| BIDV | BIDV | Ngân hàng TMCP Đầu Tư và Phát Triển Việt Nam |

**Quan trọng:** 
- Bank code phải VIẾT HOA
- Timo phải dùng code "VCCB" (không phải "BVBANK TIMO")
- VietQR API chỉ nhận đúng bank code chuẩn
